Real-Time Chat Application
📌 Overview

The Real-Time Chat Application is a full-stack web application built using Python, Django, Django Channels, WebSockets, and Django REST Framework.
It enables users to communicate through private and group chats with real-time messaging, file sharing, notifications, and secure authentication.
